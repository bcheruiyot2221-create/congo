# Brand Assets

Place official licensed logos in this folder. Supported names/formats:

- `drden-logo.svg` or `.png` or `.jpg` or `.jpeg` or `.webp`
- `drden-logo.svg` or `.png` or `.jpg` or `.jpeg` or `.webp`

The app will load these files automatically in the landing page badges and panners.
If a file is missing, it tries the next format. If none exists, it falls back to the built-in generic icon.

Recommended format: SVG with transparent background.
Recommended canvas: around 128x128 px.
